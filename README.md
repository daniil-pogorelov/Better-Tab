## üöÄ What is BetterTab?
**BetterTab** is a native macOS utility designed to replace the default `Command + Tab` switcher. It is built for developers, power users, and anyone who wants to navigate their Mac faster without touching the mouse.

Unlike **AltTab** or **Contexts**, BetterTab focuses on **keyboard-first navigation** using fuzzy search and custom bindings.

### Why replace your Mac App Switcher?
- **Stop Cycling:** Don't press `Tab` 20 times to find a window. Just type "Code" or "Spot" to jump instantly.
- **Muscle Memory:** Assign permanent hotkeys (e.g., `Option+C` for VS Code) so your apps are always one keystroke away.
- **Visual Clarity:** See exactly which window you are switching to, not just a generic app icon.
